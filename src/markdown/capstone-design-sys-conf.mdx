### Gemini API를 활용한 웹 기반의 실시간 찬반 토론 게임 구현

### 전체 구성도
![](/img/posts/capstone-design-sys-conf/sys-diagram_all.png)
- React로 웹 프론트엔드 구현
- Supabase API를 사용하여 DB, 실시간 통신 구현
- Gemini API를 사용하여 토론 내용 최종 평가

### 채팅방 생성 시
![](/img/posts/capstone-design-sys-conf/sys-diagram_create-chat.png)
1. 6자리 코드 생성
    - 채팅방 접속시 상대방이 해당 코드를 입력
2. supabase DB에 코드 등록
3. 해당 코드로 Realtime channel 생성
4. 상대방 대기

### 채팅방 접속 시
![](/img/posts/capstone-design-sys-conf/sys-diagram_join-chat.png)
1. 6자리 코드를 입력하여 채팅방 조회
2. DB에 해당 코드가 존재하면 채팅방 접속
    - 해당 코드로 Realtime channel 생성
3. 호스트에게 접속 알림
