💡소개

익명의 사용자들과 가위 바위 보 게임을 통해 가벼운 경쟁을 하는 모바일 게임입니다.

  • 개발 기간

    • (2023.08 ~ 진행 중)

  • 배경

    • 일상 속에서 소소한 재미를 느낄 수 있는 게임을 만들고 싶다는 생각에서, 누구나 간단히 즐길 수 있는 ‘가위 바위 보’를 기반으로 한 게임을 기획했습니다.

  • 목표

    • 랭킹 시스템과, 게임에서 얻는 보상을 통해 아바타를 꾸미는 기능을 구현하여, 사용자들이 지속적으로 즐길 수 있는 요소를 갖추고자 했습니다.

✨사용 기술

Flutter크로스 플랫폼을 지원하는 모바일 앱을 개발하기 위해 사용했습니다.
Supabase사용자 및 게임에 대한 데이터를 다루기 위해 사용했습니다.
FirebaseFCM(Firebase Cloud Messaging)을 통해 원격 알림 수신 기능을 구현하기 위해 사용했습니다.

💻구현

  • 가위-바위-보 게임 시스템 구현
    • 게임 요청 (송신)
      • 게임을 요청할 사용자 데이터를 DB에서 무작위로 불러오도록 구현했습니다.
      • 가위, 바위, 보 중 하나를 선택하여 해당 데이터를 DB에 저장하도록 구현했습니다.
      • 해당 사용자(수신 측) 기기에서 게임 요청 알림을 받도록 구현했습니다.
      • 게임 결과 알림을 통해 결과를 확인한 후, DB에서 해당 게임 데이터를 제거하도록 구현했습니다.
    • 게임 수락 (수신)
      • 게임 요청을 수락한 후, 가위, 바위, 보 중 하나를 선택하여 해당 데이터를 DB에 저장하도록 구현했습니다.
      • DB에 저장된 데이터에 따라 게임 결과를 출력하도록 구현했습니다.
      • 게임 결과 확인 시, 게임을 요청했던(송신 측) 사용자가 게임 결과 알림을 받도록 구현했습니다.
  • 랭킹 구현
    • 승, 패 횟수를 바탕으로 점수를 계산하도록 구현했습니다.
    • 점수를 바탕으로 전체 순위를 표시하도록 구현했습니다.
    • 상위 퍼센트에 따라 계급을 표시하도록 구현했습니다.
  • 토큰 충전 시스템 구현
    • 토큰(게임 플레이에 사용되는 화폐)을 사용하여 게임을 플레이 하도록 구현했습니다.
    • 10분 마다 1개 씩 토큰이 충전되도록 구현했습니다.
    • 토큰을 최대 5개 까지 충전할 수 있도록 구현했습니다.
  • 상점 시스템 구현
    • 게임에서 승리하거나 광고 시청으로 보석을 얻을 수 있도록 구현했습니다.
    • 보석을 사용하여 아바타를 꾸미는 기능(색상, 위치 변경 등)을 구현했습니다.
